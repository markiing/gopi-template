# gopi-template

Template de API utilizando MONGODB.
Para executar este projeto, é necessário duas bibliotecas específicas:
	go get -u github.com/gorilla/mux - "MUX Router"
	go get gopkg.in/mgo.v2 - "Mongo Driver for GO"
